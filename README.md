# CodeKeeper

<a href="https://codeclimate.com/github/SolalDR/CodeKeeper"><img src="https://codeclimate.com/github/SolalDR/CodeKeeper/badges/gpa.svg" /></a>

## Configuration

* Ruby version : 2.4.2
* Rails version : 5.0.2
* Database : postgres
